# Практический проект по дисциплине "Информационные технологии"

Данный репозиторий содержит комплекс взаимосвязанных программных модулей для загрузки, разведочного статистического анализа (EDA), адаптивной визуализации и интерактивного редактирования данных. В качестве исходного массива информации используется датасет продаж видеоигр.

Архитектура проекта и описание модулей:

*   **`datatest.py`** — Инструмент первичной подготовки и анализа данных. Выполняет аудит структуры таблицы, определяет типы данных, находит пропущенные ячейки и собирает числовую статистику. Автоматически дублирует результаты анализа в консоль и текстовый отчет `report.txt`.
*   **`data_scatter.py`** — Система базовой визуализации. Позволяет интерактивно строить двухмерные диаграммы рассеяния (Scatter Plot) с помощью кнопок выбора осей и использовать персональный маркер точек.
*   **`data_visual.py`** — Модуль улучшенной визуализации. Автоматически распознает природу данных на осях и динамически переключает тип графиков (Гистограмма, Круговая, Столбчатая и Коробочная диаграмма). Поддерживает интерактивную смену цветовых палитр (`cmap`).
*   **`data_draw.py`** — Графический редактор модификации визуализаций. Позволяет делать ручные заметки карандашом поверх построенных графиков, настраивать цвет/толщину кисти и отменять одну последнюю линию с помощью горячих клавиш `Ctrl+Z`.

*   Индивидуальные параметры:

Параметры графического интерфейса автоматически рассчитаны по правилам методических указаний:
*   **Студенческий ID:** `70225625`
*   **Маркер точечных графиков:** `>` (Треугольник вправо, расчетная свертка ID = 2)
*   **Базовая цветовая схема (`cmap`):** `Reds` (Монохромная красная палитра для первой буквы фамилии "К")
*   **Толщина линии рисования:** `6` пикселей
*   **Цвет кисти по умолчанию:** `RGB(22, 56, 25)` / HEX: `#163819` (Глубокий лесной зеленый)
